# Персональный консольный словарь

## Описание
Словарь — это простой пет-проект на Python, который позволяет хранить слова и их значения, искать их, добавлять новые и удалять существующие. Все данные сохраняются только в RAM.

## Функционал
- Поиск слова и его значения
- Добавление новых слов
- Удаление слов
- Просмотр всех слов в словаре

## Технологии и команды Python
Используются словари, списки, функции, классы, циклы и условия, обработка ошибок.

## Цель проекта
Научиться структурировать проект, работать с данными и создавать интерактивные консольные приложения на Python.
